<h4>2.管理</h4>
<h5>Wordpress信息管理</h5>
<strong>文章：</strong>一共有25篇文章。六篇月历佳作，七篇视频佳作和12篇诗集。

用户意见：2018.5.27，我的网站曾给大三新闻学生做过调研。她们认为我的文章空行过多，应当删减。并且图片容易报错（那时候我用的是图库）。

所以而根据她们的建议，我已经将所有图库都放在媒体库，并且适当删减空行。

<strong>页面：</strong>一共有10个页面。

用户意见：将原本的14级优秀诗歌词作品等分类目录移至二级菜单

<strong>标签：</strong>网站的标签是根据诗词和月历的风格、情感等元素分配的，方便用户查找自己想要看的诗集和月历。

用户意见：标签有很大的作用，将诗集的混杂情感进行了分类，便于搜索。

<strong>分类目录：</strong>网站的分类目录是根据访谈和用户调研的结果，将作品分为月历佳作，视频佳作和优秀诗词作品，并将以年级为单位存放作品。

用户意见：1.老师：以年级为单位存放

2.同学：只要我年级为二级菜单就足够了，不用再按班级分为三级目录

&nbsp;
<h5>Wordpress界面设计</h5>
<strong>选单：</strong>菜单设置了首页、月历佳作、视频佳作、优秀诗词作品、发现更多和联系我。还有四个二级菜单。

用户意见：1. 添加”发现更多“，提供用户一些古诗文网站阅览，符合甲方需求

2.添加”联系我“，添加站长信息和网站报告

<strong>侧边栏：</strong>设置了近期文章、标签、文档归类、登陆功能和搜索框。

用户意见：建议将搜索框放在侧边栏栏首，习惯”搜索框“置顶，能方便搜索。

<strong>主页互动性：</strong>主页设置了三个主要区块。三类作品都要不同呈现方式。其中，月历佳作运用了Livemesh Posts Carousel轮播形式呈现。互动性良好。

<strong>小工具：</strong>1.首页底部设有三个小工具，用户可订阅本网站

2.三个页面（月历佳作、视频佳作、优秀诗词作品）都有运用siteorigin，丰富作品呈现

3.根据用户建议，修改了侧栏的搜索框位置

&nbsp;
<h5>WORDPRESS平面设计设计</h5>
<strong>图库：</strong>1.本网站所有图库均从 PixelBay 和 Unslpash 中挑选，两大网站的图片皆是Creative Commons CC0下发布，不涉及侵权问题

用户意见：配图配的很好，与文章内容匹配度高。

配色：关于配色，我做了测试。

Anima主题预设颜色就是#D0422C，为红色。

<img class="alignnone wp-image-753 size-large" src="http://sylviatang.me/wp-content/uploads/2018/07/red-1024x546.png" alt="" width="1024" height="546" />

&nbsp;

我想了解用户对于颜色会不会有什么需求，按钮哪些颜色会让他们有点进去的欲望。由此我询问过一些同学。

发现有的用户喜欢蓝色，觉得与首页配图更匹配

&nbsp;

<img class="alignnone wp-image-751 size-large" src="http://sylviatang.me/wp-content/uploads/2018/07/blue-1024x363.png" alt="" width="1024" height="363" />

也有人喜欢绿色，觉得更有古风气息

<img class="alignnone wp-image-752 size-large" src="http://sylviatang.me/wp-content/uploads/2018/07/green-1024x524.png" alt="" width="1024" height="524" />

但综合来说，他们都认为红色是没有争议的。综上，我认为颜色其实很大一部分取决于个人喜好问题，尽量用中立没有争议的颜色，会更符合普遍用户需求。

<strong>字型字体大小：</strong>字型与字体大小我认为都是适合的。并且标题是Sans Serif 字体。

用户意见：觉得挺舒服的，可读性强，没什么问题。

<strong>排版：优秀诗词作品排版都是一个诗集一篇文章</strong>

用户意见：排版清晰，字体设置合理，空行合理，可读性强。
